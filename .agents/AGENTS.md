# Smart Campus Dataset Style Guidelines

- **Dataset Description constraint:**
  - Never describe the campus electricity dataset as "fake" or "simulated digital twin (in a way that implies it is fake)".
  - Instead, refer to it as: "A simulated operational dataset generated based on real-world campus infrastructure, occupancy patterns, laboratory usage, hostel energy demand, cafeteria usage, weather conditions, and solar energy generation."
  - All documentation, code comments, and README files must consistently refer to it as a simulated operational dataset used for developing and validating predictive machine learning models.
